# Reimbursement Calculator

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/malorydodge/reimbursement-calculator
   ```
2. CD into folder
   ```sh
   cd reimbursement-calculator
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the application
   ```sh
   node index.js
   ```
